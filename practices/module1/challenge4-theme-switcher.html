Theme Switcher Pro

Difficulty: Medium Module: State Management Concepts: StateProvider, ThemeData, Persistence Est. Time: 45-60 min

📋 Challenge Description

Build a Theme Switcher Pro — a Flutter app that allows users to toggle between Light and Dark themes with a smooth transition. The theme preference should persist even after the app is restarted, demonstrating state management combined with local persistence.

✅ Requirements

  • Light/Dark theme toggle with StateProvider
  • Theme preference persists after app restart (SharedPreferences)
  • Smooth theme transition animation
  • AppBar, background, text colors change with theme
  • Toggle button or switch with icon (sun/moon)
  • Show current theme name/status
  • Use StateProvider for theme state
  • Bonus: Add a third theme (e.g., System, Dark, Light)

📚 Resources

  • Riverpod Documentation
  • Flutter Themes
  • shared_preferences
Need a hint? Click here

💡 Hint: Use StateProvider<ThemeMode> for the theme state. Store the theme preference in SharedPreferences and load it when the app starts. Use ProviderScope with overrides to initialize theme from stored value. For the transition, wrap your MaterialApp with AnimatedTheme .

2. Theme Provider (lib/providers/theme_provider.dart)

lib/providers/theme_provider.dart import 'package:flutter/material.dart' ; import 'package:flutter_riverpod/flutter_riverpod.dart' ; import 'package:shared_preferences/shared_preferences.dart' ; final themeProvider = StateProvider<ThemeMode>((ref) { return ThemeMode.light; }); final themePreferencesProvider = FutureProvider<ThemeMode?>((ref) async { final prefs = await SharedPreferences. getInstance (); final themeString = prefs. getString ( 'theme_mode' ); if (themeString == 'dark' ) return ThemeMode.dark; if (themeString == 'light' ) return ThemeMode.light; return null ; }); final themeNotifierProvider = Provider<ThemeNotifier>((ref) { return ThemeNotifier(ref); }); class ThemeNotifier { final Ref ref; ThemeNotifier ( this .ref); void toggleTheme () { final current = ref. read (themeProvider); final newTheme = current == ThemeMode.light ? ThemeMode.dark : ThemeMode.light; ref. read (themeProvider.notifier).state = newTheme; _saveTheme (newTheme); } void setTheme (ThemeMode mode) { ref. read (themeProvider.notifier).state = mode; _saveTheme (mode); } void _saveTheme (ThemeMode mode) async { final prefs = await SharedPreferences. getInstance (); final value = mode == ThemeMode.dark ? 'dark' : 'light' ; await prefs. setString ( 'theme_mode' , value); } } // Theme data definitions final lightTheme = ThemeData( useMaterial3: true , colorScheme: ColorScheme.fromSeed( seedColor: Colors.purple, brightness: Brightness.light, ), ); final darkTheme = ThemeData( useMaterial3: true , colorScheme: ColorScheme.fromSeed( seedColor: Colors.purple, brightness: Brightness.dark, ), );

3. Main App (lib/main.dart)

lib/main.dart import 'package:flutter/material.dart' ; import 'package:flutter_riverpod/flutter_riverpod.dart' ; import 'providers/theme_provider.dart' ; import 'screens/home_screen.dart' ; void main () { runApp( const MyApp()); } class MyApp extends ConsumerStatefulWidget { const MyApp ({ super .key}); @override ConsumerState<MyApp> createState () => _MyAppState(); } class _MyAppState extends ConsumerState<MyApp> { bool _initialized = false ; ThemeMode _initialTheme = ThemeMode.light; @override void initState () { super . initState (); _loadTheme (); } void _loadTheme () async { final prefs = await SharedPreferences. getInstance (); final themeString = prefs. getString ( 'theme_mode' ); if (themeString == 'dark' ) { _initialTheme = ThemeMode.dark; } else if (themeString == 'light' ) { _initialTheme = ThemeMode.light; } setState (() { _initialized = true ; }); // Also update the provider if (mounted) { ref. read (themeProvider.notifier).state = _initialTheme; } } @override Widget build (BuildContext context) { if (!_initialized) { return MaterialApp( home: Scaffold( body: Center(child: CircularProgressIndicator()), ), ); } final themeMode = ref. watch (themeProvider); return AnimatedTheme( data: themeMode == ThemeMode.dark ? darkTheme : lightTheme, duration: Duration(milliseconds: 300 ), child: MaterialApp( title: 'Theme Switcher Pro' , theme: lightTheme, darkTheme: darkTheme, themeMode: themeMode, home: const HomeScreen(), ), ); } }
