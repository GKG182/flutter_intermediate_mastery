Async User Loader

Difficulty: Medium Module: State Management Concepts: FutureProvider, AsyncValue, HTTP Est. Time: 45-60 min

📋 Challenge Description

Build an Async User Loader — a Flutter app that fetches and displays a list of users from the JSONPlaceholder API. Use FutureProvider to manage the async data loading and AsyncValue to handle loading, error, and success states elegantly.

🌐 API Endpoint

JSONPlaceholder - Users API

https://jsonplaceholder.typicode.com/users

Returns a list of 10 users with id, name, username, email, address, phone, and website.

✅ Requirements

  • Fetch users from JSONPlaceholder API using http package
  • Use FutureProvider to manage the async data
  • Handle loading , error , and data states with AsyncValue
  • Display users in a ListView with name, email, and username
  • Add a refresh button to reload the data
  • Show a loading indicator while fetching
  • Show a user-friendly error message if the request fails
  • Use FutureProvider.autoDispose for automatic cleanup

📚 Resources

  • FutureProvider Documentation
  • http package
  • JSONPlaceholder API
  • AsyncValue Guide
Need a hint? Click here

💡 Hint: Create a User model class with fromJson factory. Use FutureProvider<List<User>> with an async function that makes the HTTP request. In the UI, use ref.watch(userProvider) and handle the AsyncValue with .when() or .whenData() to handle loading, error, and data states.

2. User Model (lib/models/user.dart)

lib/models/user.dart class User { final int id; final String name; final String username; final String email; final String phone; final String website; User ({ required this .id, required this .name, required this .username, required this .email, required this .phone, required this .website, }); factory User. fromJson (Map<String, dynamic> json) { return User( id: json[ 'id' ] as int , name: json[ 'name' ] as String, username: json[ 'username' ] as String, email: json[ 'email' ] as String, phone: json[ 'phone' ] as String, website: json[ 'website' ] as String, ); } }

3. User Provider (lib/providers/user_provider.dart)

lib/providers/user_provider.dart import 'package:flutter_riverpod/flutter_riverpod.dart' ; import 'package:http/http.dart' as http; import 'dart:convert' ; import '../models/user.dart' ; final userProvider = FutureProvider<List<User>>((ref) async { final response = await http. get ( Uri.parse( 'https://jsonplaceholder.typicode.com/users' ), ); if (response.statusCode == 200 ) { final List<dynamic> data = json. decode (response.body); return data. map ((json) => User. fromJson (json)). toList (); } else { throw Exception( 'Failed to load users: ${response.statusCode}' ); } }); // Auto-dispose version for better performance final userProviderAutoDispose = FutureProvider<List<User>>.autoDispose((ref) async { final response = await http. get ( Uri.parse( 'https://jsonplaceholder.typicode.com/users' ), ); if (response.statusCode == 200 ) { final List<dynamic> data = json. decode (response.body); return data. map ((json) => User. fromJson (json)). toList (); } else { throw Exception( 'Failed to load users: ${response.statusCode}' ); } }); // Refresh notifier final refreshProvider = StateProvider< int >((ref) => 0 );
