Async User Loader

Difficulty: Medium Module: State Management Concepts: FutureProvider, AsyncValue, HTTP Est. Time: 45-60 min

📋 Challenge Description

Build an Async User Loader — a Flutter app that fetches and displays a list of users from the JSONPlaceholder API. Use FutureProvider to manage the async data loading and AsyncValue to handle loading, error, and success states elegantly.

🌐 API Endpoint

JSONPlaceholder - Users API

https://jsonplaceholder.typicode.com/users

Returns a list of 10 users with id, name, username, email, address, phone, and website.

✅ Requirements

  • Fetch users from JSONPlaceholder API using http package
  • Use FutureProvider to manage the async data
  • Handle loading , error , and data states with AsyncValue
  • Display users in a ListView with name, email, and username
  • Add a refresh button to reload the data
  • Show a loading indicator while fetching
  • Show a user-friendly error message if the request fails
  • Use FutureProvider.autoDispose for automatic cleanup

📚 Resources

  • FutureProvider Documentation
  • http package
  • JSONPlaceholder API
  • AsyncValue Guide
Need a hint? Click here

💡 Hint: Create a User model class with fromJson factory. Use FutureProvider<List<User>> with an async function that makes the HTTP request. In the UI, use ref.watch(userProvider) and handle the AsyncValue with .when() or .whenData() to handle loading, error, and data states.

2. User Model (lib/models/user.dart)

lib/models/user.dart 
class User {
  final int id;

  final String name;

  final String username;

  final String email;

  final String phone;

  final String website;

  User({
    required this.id,

    required this.name,

    required this.username,

    required this.email,

    required this.phone,

    required this.website,
  });

  factory User.fromJson(Map<String, dynamic> json) {
    return User(
      id: json['id'] as int,
      name: json['name'] as String,
      username: json['username'] as String,
      email: json['email'] as String,
      phone: json['phone'] as String,
      website: json['website'] as String,
    );
  }
}

3. User Provider (lib/providers/user_provider.dart)

lib/providers/user_provider.dart 
import 'package:flutter_riverpod/flutter_riverpod.dart';

import 'package:http/http.dart' as http;

import 'dart:convert';

import '../models/user.dart';

final userProvider = FutureProvider<List<User>>((ref) async {
  final response = await http.get(
    Uri.parse('https://jsonplaceholder.typicode.com/users'),
  );

  if (response.statusCode == 200) {
    final List<dynamic> data = json.decode(response.body);

    return data.map((json) => User.fromJson(json)).toList();
  } else {
    throw Exception('Failed to load users: ${response.statusCode}');
  }
});

// Auto-dispose version for better performance

final userProviderAutoDispose = FutureProvider<List<User>>.autoDispose((
  ref,
) async {
  final response = await http.get(
    Uri.parse('https://jsonplaceholder.typicode.com/users'),
  );

    final List<dynamic> data = json.decode(response.body);

    return data.map((json) => User.fromJson(json)).toList();
  } else {
    throw Exception('Failed to load users: ${response.statusCode}');
  }
});

// Refresh notifier

final refreshProvider = StateProvider<int>((ref) => 0);

import 'package:flutter/material.dart';

import '../models/user.dart';

class UserCard extends StatelessWidget {
  final User user;

  const UserCard({required this.user, super.key});

  @override
  Widget build(BuildContext context) {
    return Card(
      margin: EdgeInsets.symmetric(vertical: 4),
      elevation: 2,
      shape: RoundedRectangleBorder(borderRadius: BorderRadius.circular(8)),
      child: Padding(
        padding: EdgeInsets.all(16),
        child: Row(
          children: [
            // Avatar
            CircleAvatar(
              radius: 28,
              backgroundColor:
                  Colors.primaries[user.id % Colors.primaries.length],
              child: Text(
                user.name.substring(0, 1).toUpperCase(),
                style: TextStyle(
                  color: Colors.white,
                  fontSize: 20,
                  fontWeight: FontWeight.bold,
                ),
              ),
            ),
            SizedBox(width: 16),

            // User info
            Expanded(
              child: Column(
                crossAxisAlignment: CrossAxisAlignment.start,
                children: [
                  Text(
                    user.name,
                    style: TextStyle(fontSize: 16, fontWeight: FontWeight.bold),
                  ),
                  Text(
                    '@${user.username}',
                    style: TextStyle(fontSize: 12, color: Colors.grey[600]),
                  ),
                  SizedBox(height: 4),
                  Row(
                    children: [
                      Icon(Icons.email, size: 14, color: Colors.grey[500]),
                      SizedBox(width: 4),
                      Text(
                        user.email,
                        style: TextStyle(fontSize: 12, color: Colors.grey[600]),
                      ),
                    ],
                  ),
                  Row(
                    children: [
                      Icon(Icons.phone, size: 14, color: Colors.grey[500]),

5. Main Screen (lib/main.dart)

lib/main.dart 
import 'package:flutter/material.dart';

import 'package:flutter_riverpod/flutter_riverpod.dart';

import 'providers/user_provider.dart';

import 'widgets/user_card.dart';

void main() {
  runApp(const ProviderScope(child: MyApp()));
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Async User Loader',
      theme: ThemeData(
        colorScheme: ColorScheme.fromSeed(seedColor: Colors.purple),
        useMaterial3: true,
      ),
      home: const HomeScreen(),
    );
  }
}

class HomeScreen extends ConsumerWidget {
  const HomeScreen({super.key});

  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final usersAsync = ref.watch(userProvider);

    return Scaffold(
      appBar: AppBar(
        title: Text('Async User Loader'),
        backgroundColor: Colors.purple,
        foregroundColor: Colors.white,
        actions: [
          IconButton(
            onPressed: () {
              // Refresh by invalidating the provider

              ref.invalidate(userProvider);
            },
            icon: Icon(Icons.refresh),
            tooltip: 'Refresh',
          ),
        ],
      ),
      body: usersAsync.when(
        loading: () => Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              CircularProgressIndicator(),
              SizedBox(height: 16),
              Text(
                'Loading users...',
                style: TextStyle(color: Colors.grey[600]),
              ),
            ],
          ),
        ),
        error: (error, stack) => Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              Icon(Icons.error_outline, size: 64, color: Colors.red[300]),
              SizedBox(height: 16),
              Text(
                'Something went wrong',
                style: TextStyle(fontSize: 18, fontWeight: FontWeight.bold),
              ),
              SizedBox(height: 8),
              Text(
                error.toString(),
                textAlign: TextAlign.center,
                style: TextStyle(color: Colors.grey[600]),
              ),
              SizedBox(height: 16),
              ElevatedButton(
                onPressed: () {
                  ref.invalidate(userProvider);
                },
                child: Text('Try Again'),
              ),
            ],
          ),
        ),
        data: (users) {
          return ListView.builder(
            padding: EdgeInsets.all(16),
            itemCount: users.length,
            itemBuilder: (context, index) {
              return UserCard(user: users[index]);
            },
          );
        },
      ),
    );
  }
}
