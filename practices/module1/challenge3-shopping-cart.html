Shopping Cart Calculator

Difficulty: Easy Module: State Management Concepts: StateNotifierProvider, Derived State Est. Time: 30-45 min

📋 Challenge Description

Build a Shopping Cart Calculator — a simple e-commerce cart that demonstrates derived state and total calculation. Users can browse products, add them to the cart, adjust quantities, remove items, and see the total update automatically.

✅ Requirements

  • Display a list of products with name and price
  • Each product has an "Add to Cart" button
  • The cart shows added items with quantities
  • Each cart item has increment (+) and decrement (-) buttons
  • Each cart item has a remove (×) button
  • Display the subtotal , tax (10%), and total
  • Show the number of items in the cart badge
  • Use StateNotifier and StateNotifierProvider
  • Use Provider for derived state (total, tax, item count)

📚 Resources

  • StateNotifierProvider
  • Riverpod Providers
  • Flutter Cookbook
Need a hint? Click here

💡 Hint: Define a CartItem class with product and quantity . Use StateNotifier<List<CartItem>> for the cart. Use Provider to calculate derived values like totalItems (sum of quantities) and totalPrice (sum of price * quantity ).

4. Cart Provider (lib/providers/cart_provider.dart)

lib/providers/cart_provider.dart import 'package:flutter_riverpod/flutter_riverpod.dart' ; import '../models/cart_item.dart' ; import '../models/product.dart' ; class CartNotifier extends StateNotifier<List<CartItem>> { CartNotifier () : super ([]); void addItem (Product product) { final existingIndex = state. indexWhere ( (item) => item.product.id == product.id, ); if (existingIndex != - 1 ) { // Product already in cart, increment quantity final updatedItem = state[existingIndex]; state = [ ...state. sublist ( 0 , existingIndex), CartItem( product: updatedItem.product, quantity: updatedItem.quantity + 1 , ), ...state. sublist (existingIndex + 1 ), ]; } else { // New product state = [...state, CartItem(product: product)]; } } void removeItem (String productId) { state = state. where ((item) => item.product.id != productId). toList (); } void updateQuantity (String productId, int quantity) { final index = state. indexWhere ((item) => item.product.id == productId); if (index != - 1 ) { if (quantity <= 0 ) { removeItem (productId); } else { state = [ ...state. sublist ( 0 , index), CartItem( product: state[index].product, quantity: quantity, ), ...state. sublist (index + 1 ), ]; } } } void clearCart () { state = []; } } // Providers final cartProvider = StateNotifierProvider<CartNotifier, List<CartItem>>((ref) { return CartNotifier(); }); // Derived state: total number of items final totalItemsProvider = Provider< int >((ref) { final cart = ref. watch (cartProvider); return cart. fold ( 0 , (sum, item) => sum + item.quantity); }); // Derived state: subtotal final subtotalProvider = Provider< double >((ref) { final cart = ref. watch (cartProvider); return cart. fold ( 0.0 , (sum, item) => sum + item.totalPrice); }); // Derived state: tax (10%) final taxProvider = Provider< double >((ref) { final subtotal = ref. watch (subtotalProvider); return subtotal * 0.10 ; }); // Derived state: total final totalProvider = Provider< double >((ref) { final subtotal = ref. watch (subtotalProvider); final tax = ref. watch (taxProvider); return subtotal + tax; });
