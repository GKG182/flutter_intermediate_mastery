Live Counter Stream

Difficulty: Medium Module: State Management Concepts: StreamProvider, AsyncValue, Stream Est. Time: 45-60 min

📋 Challenge Description

Build a Live Counter Stream — a Flutter app that displays a counter that automatically increments every second. Use StreamProvider to create and manage a stream of counter values, and AsyncValue to handle the stream states elegantly.

✅ Requirements

  • Create a Stream that emits a new counter value every second
  • Use StreamProvider to provide the stream to the UI
  • Display the live counter value prominently
  • Show a timer indicator (e.g., "Ticking every second")
  • Add Pause and Resume controls
  • Use AsyncValue to handle the stream data
  • Show the elapsed time or ticks count
  • Add a Reset button to start over

📚 Resources

  • StreamProvider Documentation
  • Dart Streams
  • Stream.periodic
  • AsyncValue Guide
Need a hint? Click here

💡 Hint: Use Stream.periodic(Duration(seconds: 1), (count) => count) to create a counter stream. Use StreamProvider<int> to expose it. For controls, use a StateProvider<bool> to track if the stream is paused. Use a StreamController with a broadcast for manual control over the stream.

2. Counter Provider (lib/providers/counter_provider.dart)

lib/providers/counter_provider.dart 
import 'package:flutter_riverpod/flutter_riverpod.dart';

import 'dart:async';

// Stream provider that emits a counter every second

final counterStreamProvider = StreamProvider<int>((ref) {
  // Listen for pause state

  final isPaused = ref.watch(pauseProvider);

  // Create a controller that we can manually control

  final controller = StreamController<int>();

  int counter = 0;

  // Timer for periodic updates

  Timer? timer;

  void startTimer() {
    timer = Timer.periodic(Duration(seconds: 1), (t) {
      if (!controller.isClosed) {
        counter++;
        controller.add(counter);
      }
    });
  }

  // Start the timer

  startTimer();

  // Handle pause/resume

  late bool previousPaused = isPaused;

  // Watch for pause changes

  final pauseListener = ref.listen(pauseProvider, (previous, next) {
    if (next && !previous) {
      // Paused

      timer?.cancel();
    } else if (!next && previous) {
      // Resumed

      startTimer();
    }
    previous = next;
  });

  // Cleanup on dispose

  ref.onDispose(() {
    timer?.cancel();
    controller.close();
  });

  return controller.stream.asBroadcastStream();
});

// Pause state provider

final pauseProvider = StateProvider<bool>((ref) => false);

// Reset function

final resetProvider = Provider<VoidCallback>((ref) {
  return () {
    // Reset the counter by invalidating the stream provider

    ref.invalidate(counterStreamProvider);
  };
});

3. Main Screen (lib/main.dart)

lib/main.dart 
import 'package:flutter/material.dart';

import 'package:flutter_riverpod/flutter_riverpod.dart';

import 'providers/counter_provider.dart';

void main() {
  runApp(const ProviderScope(child: MyApp()));
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Live Counter Stream',
      theme: ThemeData(
        colorScheme: ColorScheme.fromSeed(seedColor: Colors.orange),
        useMaterial3: true,
      ),
      home: const HomeScreen(),
    );
  }
}

class HomeScreen extends ConsumerWidget {
  const HomeScreen({super.key});

  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final counterAsync = ref.watch(counterStreamProvider);

    final isPaused = ref.watch(pauseProvider);

    return Scaffold(
      appBar: AppBar(
        title: Text('Live Counter'),
        backgroundColor: Colors.orange,
        foregroundColor: Colors.white,
        actions: [
          IconButton(
            onPressed: () {
              ref.read(pauseProvider.notifier).state = !isPaused;
            },
            icon: Icon(isPaused ? Icons.play_arrow : Icons.pause),
            tooltip: isPaused ? 'Resume' : 'Pause',
          ),
          IconButton(
            onPressed: () {
              ref.invalidate(counterStreamProvider);
            },
            icon: Icon(Icons.refresh),
            tooltip: 'Reset',
          ),
        ],
      ),
      body: counterAsync.when(
        loading: () => Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              CircularProgressIndicator(),
              SizedBox(height: 16),
              Text(
                'Starting counter...',
                style: TextStyle(color: Colors.grey[600]),
              ),
            ],
          ),
        ),
        error: (error, stack) => Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              Icon(Icons.error_outline, size: 64, color: Colors.red[300]),
              SizedBox(height: 16),
              Text(
                'Stream error',
                style: TextStyle(fontSize: 18, fontWeight: FontWeight.bold),
              ),
              SizedBox(height: 8),
              Text(error.toString(), style: TextStyle(color: Colors.grey[600])),
            ],
          ),
        ),
        data: (counter) {
          return Center(
            child: Column(
              mainAxisAlignment: MainAxisAlignment.center,
              children: [
                // Status indicator
                Container(
                  padding: EdgeInsets.symmetric(horizontal: 16, vertical: 6),
                  decoration: BoxDecoration(
                    color: isPaused ? Colors.orange[100] : Colors.green[100],
                    borderRadius: BorderRadius.circular(20),
                  ),
                  child: Row(
                    mainAxisSize: MainAxisSize.min,
                    children: [
                      Container(
                        width: 10,
                        height: 10,
                        decoration: BoxDecoration(
                          color: isPaused ? Colors.orange : Colors.green,
                          shape: BoxShape.circle,
                        ),
                      ),
                      SizedBox(width: 8),
                      Text(
                        isPaused ? '⏸️ Paused' : '▶️ Live',
                        style: TextStyle(
                          fontWeight: FontWeight.w600,
                          color: isPaused
                              ? Colors.orange[800]
                              : Colors.green[800],
                        ),
                      ),
                    ],
                  ),
                ),
                SizedBox(height: 48),

                // Counter display
                Stack(
                  alignment: Alignment.center,
                  children: [
                    Container(
                      width: 200,
                      height: 200,
                      decoration: BoxDecoration(
                        shape: BoxShape.circle,
                        gradient: LinearGradient(
                          colors: [Colors.orange[100]!, Colors.orange[300]!],
                        ),
                      ),
                      child: Center(
                        child: Text(
                          counter.toString(),
                          style: TextStyle(
                            fontSize: 72,
                            fontWeight: FontWeight.bold,
                            color: Colors.orange[900],
                          ),
                        ),
                      ),
                    ),

                    // Pulse animation indicator
                    if (!isPaused)
                      AnimatedContainer(
                        duration: Duration(seconds: 1),
                        width: 220,
                        height: 220,
                        decoration: BoxDecoration(
                          shape: BoxShape.circle,
                          border: Border.all(
                            color: Colors.orange.withOpacity(0.3),
                            width: 4,
                          ),
                        ),
                      ),
                  ],
                ),
                SizedBox(height: 32),

                // Timer info
                Text(
                  '⏱️ Ticking every second',
                  style: TextStyle(fontSize: 14, color: Colors.grey[600]),
                ),
                SizedBox(height: 8),
                Text(
                  'Counter: $counter',
                  style: TextStyle(fontSize: 16, fontWeight: FontWeight.w500),
                ),
                SizedBox(height: 32),

                // Controls
                Row(
                  mainAxisAlignment: MainAxisAlignment.center,
                  children: [
                    ElevatedButton.icon(
                      onPressed: () {
                        ref.read(pauseProvider.notifier).state = !isPaused;
                      },
                      icon: Icon(isPaused ? Icons.play_arrow : Icons.pause),
                      label: Text(isPaused ? 'Resume' : 'Pause'),
                      style: ElevatedButton.styleFrom(
                        backgroundColor: isPaused
                            ? Colors.green
                            : Colors.orange,
                        foregroundColor: Colors.white,
                      ),
                    ),
                    SizedBox(width: 12),
                    OutlinedButton.icon(
                      onPressed: () {
                        ref.invalidate(counterStreamProvider);
                      },
                      icon: Icon(Icons.refresh),
                      label: Text('Reset'),
                    ),
                  ],
                ),
                SizedBox(height: 16),

                // Info text
                Text(
                  isPaused
                      ? '⏸️ Counter paused at $counter'
                      : '⏳ Counting... $counter',
                  style: TextStyle(
                    fontSize: 12,
                    color: Colors.grey[500],
                    fontStyle: FontStyle.italic,
                  ),
                ),
              ],
            ),
          );
        },
      ),
    );
  }
}

✅ Key Concepts Demonstrated

  • StreamProvider – Managing real-time data streams
  • Stream.periodic – Creating a timer-based stream
  • AsyncValue – Handling loading/error/data states
  • StreamController – Manual control over streams
  • Pause/Resume – Controlling the stream with StateProvider
  • Broadcast Stream – Allowing multiple listeners
