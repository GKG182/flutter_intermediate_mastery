Live Counter Stream

Difficulty: Medium Module: State Management Concepts: StreamProvider, AsyncValue, Stream Est. Time: 45-60 min

📋 Challenge Description

Build a Live Counter Stream — a Flutter app that displays a counter that automatically increments every second. Use StreamProvider to create and manage a stream of counter values, and AsyncValue to handle the stream states elegantly.

✅ Requirements

  • Create a Stream that emits a new counter value every second
  • Use StreamProvider to provide the stream to the UI
  • Display the live counter value prominently
  • Show a timer indicator (e.g., "Ticking every second")
  • Add Pause and Resume controls
  • Use AsyncValue to handle the stream data
  • Show the elapsed time or ticks count
  • Add a Reset button to start over

📚 Resources

  • StreamProvider Documentation
  • Dart Streams
  • Stream.periodic

Need a hint? Click here

💡 Hint: Use Stream.periodic(Duration(seconds: 1), (count) => count) to create a counter stream. Use StreamProvider<int> to expose it. For controls, use a StateProvider<bool> to track if the stream is paused. Use a StreamController with a broadcast for manual control over the stream.

1. pubspec.yaml

2. Counter Provider (lib/providers/counter_provider.dart)

lib/providers/counter_provider.dart import 'package:flutter_riverpod/flutter_riverpod.dart'; import 'dart:async'; // Stream provider that emits a counter every second final counterStreamProvider = StreamProvider<int>((ref) { // Listen for pause state final isPaused = ref.watch(pauseProvider); // Create a controller that we can manually control final controller = StreamController<int>(); int counter = 0; // Timer for periodic updates Timer? timer; void startTimer() { timer = Timer.periodic(Duration(seconds: 1), (t) { if (!controller.isClosed) { counter++; controller.add(counter); } }); } // Start the timer startTimer(); // Handle pause/resume late bool previousPaused = isPaused; // Watch for pause changes final pauseListener = ref.listen(pauseProvider, (previous, next) { if (next && !previous) { // Paused timer?.cancel(); } else if (!next && previous) { // Resumed startTimer(); } previous = next; }); // Cleanup on dispose ref.onDispose(() { timer?.cancel(); controller.close(); }); return controller.stream.asBroadcastStream(); }); // Pause state provider final pauseProvider = StateProvider<bool>((ref) => false); // Reset function final resetProvider = Provider<VoidCallback>((ref) { return () { // Reset the counter by invalidating the stream provider ref.invalidate(counterStreamProvider); }; });

  • AsyncValue – Handling loading/error/data states
  • StreamController – Manual control over streams
  • Pause/Resume – Controlling the stream with StateProvider
  • Broadcast Stream – Allowing multiple listeners
