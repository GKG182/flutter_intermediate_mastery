Multi-Step Form State Manager

Difficulty: Hard Module: State Management Concepts: StateNotifier, Multi-Step Forms, Navigation Est. Time: 60-90 min

📋 Challenge Description

Build a Multi-Step Form State Manager — a Flutter app that guides users through a 3-step form flow (Personal Info → Address → Review & Submit) while preserving state across screens. Use StateNotifier to manage the form state and navigation to move between steps.

✅ Requirements

  • 3-step form flow:
    • Step 1: Personal Info (Name, Email, Phone)
    • Step 2: Address (Street, City, ZIP, Country)
    • Step 3: Review & Submit (show all data)
  • State persistence across all steps
  • Navigation between steps (Next/Back buttons)
  • Step indicator showing progress (e.g., Step 1/3)
  • Validation on each step before proceeding
  • Review screen showing all collected data
  • Submit with success message
  • Use StateNotifier for form state management
  • Clean, responsive UI with proper styling

📚 Resources

  • StateNotifierProvider
  • Flutter Navigation
  • flutter_riverpod
Need a hint? Click here

💡 Hint: Create a FormData class with all form fields. Use StateNotifier<FormData> to manage the state. For navigation, use Navigator.push to navigate to each step screen. Each screen should receive the FormData and update it using the notifier. Use a StateProvider<int> to track the current step.

2. Form Data Model (lib/models/form_data.dart)

lib/models/form_data.dart class FormData { // Step 1: Personal Info final String name; final String email; final String phone; // Step 2: Address final String street; final String city; final String zipCode; final String country; FormData ({ this .name = '' , this .email = '' , this .phone = '' , this .street = '' , this .city = '' , this .zipCode = '' , this .country = '' , }); FormData copyWith ({ String? name, String? email, String? phone, String? street, String? city, String? zipCode, String? country, }) { return FormData( name: name ?? this .name, email: email ?? this .email, phone: phone ?? this .phone, street: street ?? this .street, city: city ?? this .city, zipCode: zipCode ?? this .zipCode, country: country ?? this .country, ); } bool get isStep1Valid => name. isNotEmpty && email. contains ( '@' ) && phone. isNotEmpty ; bool get isStep2Valid => street. isNotEmpty && city. isNotEmpty && zipCode. isNotEmpty && country. isNotEmpty ; bool get isComplete => isStep1Valid && isStep2Valid; }

3. Form Provider (lib/providers/form_provider.dart)

lib/providers/form_provider.dart import 'package:flutter_riverpod/flutter_riverpod.dart' ; import '../models/form_data.dart' ; class FormNotifier extends StateNotifier<FormData> { FormNotifier () : super (FormData()); // Step 1: Personal Info void updateName (String value) { state = state. copyWith (name: value); } void updateEmail (String value) { state = state. copyWith (email: value); } void updatePhone (String value) { state = state. copyWith (phone: value); } // Step 2: Address void updateStreet (String value) { state = state. copyWith (street: value); } void updateCity (String value) { state = state. copyWith (city: value); } void updateZipCode (String value) { state = state. copyWith (zipCode: value); } void updateCountry (String value) { state = state. copyWith (country: value); } void resetForm () { state = FormData(); } } final formProvider = StateNotifierProvider<FormNotifier, FormData>((ref) { return FormNotifier(); }); final currentStepProvider = StateProvider< int >((ref) => 0 );
