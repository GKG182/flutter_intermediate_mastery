Form Validation System

Difficulty: Medium Module: State Management Concepts: StateNotifier, Form Validation, Password Strength Est. Time: 45-60 min

📋 Challenge Description

Build a Form Validation System — a Flutter app with a login/signup form that features real-time email validation, a password strength meter, and comprehensive form validation. Use StateNotifier to manage the form state and validation logic.

✅ Requirements

  • Email field with real-time validation (valid format)
  • Password field with strength meter (Weak/Medium/Strong)
  • Confirm Password field with match validation
  • Submit button that is enabled only when all fields are valid
  • Real-time validation feedback with error messages
  • Password strength criteria:
    • Weak: < 6 characters
    • Medium: 6-9 characters
    • Strong: 10+ characters with letters, numbers, and symbols
  • Use StateNotifier for form state management
  • Show success message on valid submission
  • Clean, responsive UI with proper styling

📚 Resources

  • StateNotifierProvider
  • Flutter Form Validation
  • flutter_riverpod
Need a hint? Click here

💡 Hint: Create a FormState class with fields for email, password, confirmPassword, and validation errors. Use StateNotifier to update the state as the user types. For password strength, check length and character types (uppercase, lowercase, numbers, symbols). Use a Provider to compute the form validity based on the current state.

3. Form Notifier (lib/providers/form_provider.dart)

lib/providers/form_provider.dart import 'package:flutter_riverpod/flutter_riverpod.dart' ; import '../models/form_state.dart' ; class FormNotifier extends StateNotifier<FormState> { FormNotifier () : super (FormState()); void updateEmail (String value) { state = state. copyWith ( email: value, emailError: _validateEmail (value), isSuccess: false , isSubmitted: false , ); } void updatePassword (String value) { state = state. copyWith ( password: value, passwordError: _validatePassword (value), isSuccess: false , isSubmitted: false , ); // Re-validate confirm password if it has a value if (state.confirmPassword. isNotEmpty ) { _validateConfirmPassword (state.confirmPassword); } } void updateConfirmPassword (String value) { state = state. copyWith ( confirmPassword: value, confirmPasswordError: _validateConfirmPassword (value), isSuccess: false , isSubmitted: false , ); } String? _validateEmail (String value) { if (value. isEmpty ) { return 'Email is required' ; } final emailRegex = RegExp( r'^[\w-\.]+@([\w-]+\.)+[\w-]{2,4}$' , ); if (!emailRegex. hasMatch (value)) { return 'Please enter a valid email address' ; } return null ; } String? _validatePassword (String value) { if (value. isEmpty ) { return 'Password is required' ; } if (value. length < 6 ) { return 'Password must be at least 6 characters' ; } return null ; } String? _validateConfirmPassword (String value) { if (value. isEmpty ) { return 'Please confirm your password' ; } if (value != state.password) { return 'Passwords do not match' ; } return null ; } void submitForm () { // Re-validate all fields final emailError = _validateEmail (state.email); final passwordError = _validatePassword (state.password); final confirmError = _validateConfirmPassword (state.confirmPassword); state = state. copyWith ( emailError: emailError, passwordError: passwordError, confirmPasswordError: confirmError, isSubmitted: true , ); if (state.isValid) { // Form is valid - submit it state = state. copyWith (isSuccess: true ); } } void resetForm () { state = FormState(); } String getPasswordStrength (String password) { if (password. isEmpty ) return '' ; if (password. length < 6 ) return 'Weak' ; if (password. length < 10 ) return 'Medium' ; bool hasLetter = password. contains (RegExp( r'[A-Za-z]' )); bool hasNumber = password. contains (RegExp( r'[0-9]' )); bool hasSymbol = password. contains (RegExp( r'[!@#$%^&*(),.?":{}|<>]' )); if (hasLetter && hasNumber && hasSymbol) return 'Strong' ; if (hasLetter && hasNumber) return 'Medium' ; return 'Weak' ; } Color getPasswordStrengthColor (String password) { final strength = getPasswordStrength (password); switch (strength) { case 'Weak' : return Colors.red; case 'Medium' : return Colors.orange; case 'Strong' : return Colors.green; default : return Colors.grey; } } } final formProvider = StateNotifierProvider<FormNotifier, FormState>((ref) { return FormNotifier(); });
