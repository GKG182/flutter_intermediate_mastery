Realtime Chat App

Difficulty: Hard Module: Supabase Concepts: Supabase Realtime, Streams, Subscriptions, Authentication Est. Time: 50-60 min 🗄️ Supabase Realtime

📋 Challenge Description

Build a Realtime Chat App using Supabase Realtime subscriptions. Users can send and receive messages instantly without refreshing, with proper authentication and message history.

✅ Requirements

  • Use Supabase Realtime for live messaging
  • Use Supabase Auth for user authentication (from previous challenge)
  • Create a messages table with id , content , user_id , username , created_at
  • Enable Realtime on the messages table
  • Subscribe to new messages with channel
  • Display messages in a ListView with newest at bottom
  • Add a message input field with send button
  • Show user avatars with initials
  • Add timestamps to messages
  • Auto-scroll to latest message
  • Handle typing indicators (optional but encouraged)
  • Show online status (optional but encouraged)

🗄️ Supabase Realtime Setup

Supabase Realtime uses WebSockets for live data synchronization.

-- Enable Realtime on your messages table in Supabase Dashboard
ALTER TABLE messages REPLICA IDENTITY FULL;
💡 Note: You must enable Realtime on the messages table in the Supabase Dashboard > Database > Replication.

📚 Resources

  • Supabase Flutter Package
  • Supabase Realtime Docs
  • Flutter Realtime Guide

💡 Hint: Use supabase.channel() to create a realtime channel. Subscribe to postgres_changes with event: 'INSERT' and schema: 'public'. Use StreamProvider to listen to the subscription stream. For the message list, use StreamBuilder or ListView.builder with a stream from the channel. Don't forget to dispose() the channel in the provider.

3. Message Model (lib/models/message.dart)

lib/models/message.dart 
import 'package:intl/intl.dart';

class Message {
  final String id;

  final String content;

  final String userId;

  final String username;

  final DateTime createdAt;

  Message({
    required this.id,

    required this.content,

    required this.userId,

    required this.username,

    required this.createdAt,
  });

  factory Message.fromJson(Map<String, dynamic> json) {
    return Message(
      id: json['id'],
      content: json['content'],
      userId: json['user_id'],
      username: json['username'],
      createdAt: DateTime.parse(json['created_at']),
    );
  }

  Map<String, dynamic> toJson() {
    return {
      'id': id,

      'content': content,

      'user_id': userId,

      'username': username,

      'created_at': createdAt.toIso8601String(),
    };
  }

  factory Message.forInsert({
    required String content,

    required String userId,

    required String username,
  }) {
    return Message(
      id: '',
      content: content,
      userId: userId,
      username: username,
      createdAt: DateTime.now(),
    );
  }

  String getFormattedTime() {
    final now = DateTime.now();

    final difference = now.difference(createdAt);

    if (difference.inDays > 0) {
      return DateFormat.yMMMd().add_jm().format(createdAt);
    } else {
      return DateFormat.jm().format(createdAt);
    }
  }

  bool isFromCurrentUser(String currentUserId) {
    return userId == currentUserId;
  }

  String getInitials() {
    return username
        .split(' ')
        .map((word) => word.isNotEmpty ? word[0] : '')
        .join()
        .toUpperCase()
        .substring(0, 2);
  }
}

4. Chat Provider (lib/providers/chat_provider.dart)

import 'dart:async';

import 'package:flutter_riverpod/flutter_riverpod.dart';

import 'package:supabase_flutter/supabase_flutter.dart';

import '../models/message.dart';

final supabase = Supabase.instance.client;

// Provider for the current user's username

final currentUsernameProvider = Provider<String?>((ref) {
  final user = supabase.auth.currentUser;

  return user?.userMetadata?['username'] as String?;
});

// Chat state provider with realtime subscription

final chatProvider = StreamProvider<List<Message>>((ref) {
  return _getChatStream(ref);
});

Stream<List<Message>> _getChatStream(Ref ref) async* {
  // First, load all existing messages

  final initialMessages = await _loadMessages();
  List<Message> messages = initialMessages;

  // Create a channel for realtime updates

  final channel = supabase.channel('public:messages');

  // Create a controller for new messages

  final controller = StreamController<List<Message>>();

  // Emit initial messages

  controller.add(messages);

  // Listen for new messages

  channel.on(
    RealtimeListenTypes.postgresChanges,
    ChannelFilter(event: 'INSERT', schema: 'public', table: 'messages'),
    (payload, [ref]) {
      final newMessage = Message.fromJson(
        payload['new'] as Map<String, dynamic>,
      );
      messages = [newMessage, ...messages];
      controller.add(messages);
    },
  );

  // Subscribe to the channel

  await channel.subscribe();

  // Yield messages as they come

  await for (final _ in controller.stream) {
    yield messages;
  }

  // Clean up when the provider is disposed

  ref.onDispose(() {
    controller.close();
    supabase.removeChannel(channel);
  });
}

Future<List<Message>> _loadMessages() async {
  final response = await supabase
      .from('messages')
      .select('*')
      .order('created_at', ascending: false)
      .limit(100);

  return (response as List).map((json) => Message.fromJson(json)).toList();
}

// Send a new message

final sendMessageProvider = Provider<void Function(String)>((ref) {
  return _sendMessage;
});

void _sendMessage(String content) async {
  final user = supabase.auth.currentUser;

  if (user == null || content.trim().isEmpty) return;

  final username = user.userMetadata?['username'] as String? ?? 'Anonymous';

  await supabase.from('messages').insert({
    'content': content.trim(),

    'user_id': user.id,

    'username': username,
  });
}

import 'package:flutter_riverpod/flutter_riverpod.dart';

import 'package:supabase_flutter/supabase_flutter.dart';

import 'package:google_sign_in/google_sign_in.dart';

final supabase = Supabase.instance.client;

final authStateProvider = StreamProvider<User?>((ref) {
  return supabase.auth.onAuthStateChange().map((event) {
    return event.session?.user;
  });
});

final currentUserProvider = Provider<User?>((ref) {
  return supabase.auth.currentUser;
});

class AuthNotifier extends StateNotifier<AsyncValue<User?>> {
  AuthNotifier() : super(const AsyncValue.loading()) {
    _loadInitialUser();
  }

  void _loadInitialUser() {
    final user = supabase.auth.currentUser;
    state = AsyncValue.data(user);
  }

  Future<void> signUp({
    required String email,

    required String password,

    required String username,
  }) async {
    state = const AsyncValue.loading();

    try {
      final response = await supabase.auth.signUp(
        email: email,
        password: password,
        data: {'username': username},
      );
      state = AsyncValue.data(response.user);
    } catch (e) {
      state = AsyncValue.error(e.toString(), StackTrace.current);

      rethrow;
    }
  }

  Future<void> signIn({required String email, required String password}) async {
    state = const AsyncValue.loading();

    try {
      final response = await supabase.auth.signInWithPassword(
        email: email,
        password: password,
      );
      state = AsyncValue.data(response.user);
    } catch (e) {
      state = AsyncValue.error(e.toString(), StackTrace.current);

      rethrow;
    }
  }

  Future<void> signInWithGoogle() async {
    state = const AsyncValue.loading();

    try {
      final GoogleSignIn googleSignIn = GoogleSignIn(
        serverClientId: 'YOUR_GOOGLE_WEB_CLIENT_ID',
      );

      final googleUser = await googleSignIn.signIn();

      if (googleUser == null) throw Exception('Google sign-in cancelled');

      final googleAuth = await googleUser.authentication;

      final response = await supabase.auth.signInWithIdToken(
        provider: OAuthProvider.google,
        idToken: googleAuth.idToken!,
        accessToken: googleAuth.accessToken,
      );
      state = AsyncValue.data(response.user);
    } catch (e) {
      state = AsyncValue.error(e.toString(), StackTrace.current);

      rethrow;
    }
  }

  Future<void> signOut() async {
    state = const AsyncValue.loading();

    try {
      await GoogleSignIn().signOut();

      await supabase.auth.signOut();
      state = const AsyncValue.data(null);
    } catch (e) {
      state = AsyncValue.error(e.toString(), StackTrace.current);

      rethrow;
    }
  }
}

final authNotifierProvider =
    StateNotifierProvider<AuthNotifier, AsyncValue<User?>>((ref) {
      return AuthNotifier();
    });

                      child: Text(
                        'OR',
                        style: TextStyle(color: Colors.grey[400]),
                      ),
                    ),
                    Expanded(child: Divider()),
                  ],
                ),
                SizedBox(height: 16),
                OutlinedButton.icon(
                  onPressed: _isLoading ? null : _handleGoogleSignIn,
                  icon: Image.network(
                    'https://www.gstatic.com/firebasejs/ui/2.0.0/images/auth/google.svg',
                    width: 24,
                    height: 24,
                  ),
                  label: Text('Continue with Google'),
                  style: OutlinedButton.styleFrom(
                    padding: EdgeInsets.symmetric(vertical: 14),
                    shape: RoundedRectangleBorder(
                      borderRadius: BorderRadius.circular(8),
                    ),
                  ),
                ),
                SizedBox(height: 24),
                Row(
                  mainAxisAlignment: MainAxisAlignment.center,
                  children: [
                    Text(
                      _isLogin
                          ? "Don't have an account?"
                          : 'Already have an account?',
                      style: TextStyle(color: Colors.grey[600]),
                    ),
                    TextButton(
                      onPressed: _isLoading ? null : _toggleMode,
                      child: Text(
                        _isLogin ? 'Sign Up' : 'Sign In',
                        style: TextStyle(
                          color: Colors.blue,
                          fontWeight: FontWeight.bold,
                        ),
                      ),
                    ),
                  ],
                ),
              ],
            ),
          ),
        ),
      ),
    );
  }

  • StreamProvider – Handling realtime streams with Riverpod
  • Database Policies – Row Level Security for message access
  • Channel Subscriptions – Listening to INSERT events on the messages table
  • Auto-scroll – Automatically scrolling to latest messages
  • Message Bubbles – Differentiating current user vs other users
