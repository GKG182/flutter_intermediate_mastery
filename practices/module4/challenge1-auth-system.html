Authentication System

Difficulty: Medium Module: Supabase Concepts: Supabase Auth, Email/Password, OAuth, Session Management Est. Time: 40-50 min 🗄️ Supabase Auth

📋 Challenge Description

Build an Authentication System using Supabase Auth that supports email/password sign-up and sign-in, Google OAuth, and session persistence. Users should be able to manage their profile and sign out securely.

✅ Requirements

  • Use Supabase for authentication (not Firebase)
  • Implement email/password sign-up with user metadata
  • Implement email/password sign-in
  • Add Google OAuth sign-in
  • Implement session persistence (auto-login)
  • Listen to auth state changes with Riverpod
  • Show protected dashboard after login
  • Display user profile information (email, metadata)
  • Implement sign out functionality
  • Show loading states during authentication
  • Handle errors gracefully (invalid credentials, network issues)

🗄️ Supabase Setup

Supabase provides a complete authentication system with built-in session management.

flutter pub add supabase_flutter google_sign_in sign_in_with_apple
💡 Note: You'll need to set up a Supabase project and enable email/password and Google providers in the dashboard.

📚 Resources

  • Supabase Flutter Package
  • Supabase Auth Docs
  • Google Sign-In
  • Riverpod Documentation
Need a hint? Click here

💡 Hint: Use Supabase.initialize() in main.dart with your project URL and anon key. Create a Riverpod provider that listens to supabase.auth.onAuthStateChange. Use signUp() and signInWithPassword() for email auth. For Google, use signInWithIdToken() with the provider set to OAuthProvider.google. Use currentUser and currentSession to check auth state.

3. Auth Provider (lib/providers/auth_provider.dart)

lib/providers/auth_provider.dart import 'package:flutter_riverpod/flutter_riverpod.dart'; import 'package:supabase_flutter/supabase_flutter.dart'; import 'package:google_sign_in/google_sign_in.dart'; final supabase = Supabase.instance.client; // Auth state provider - listens to auth changes final authStateProvider = StreamProvider<User?>((ref) { return supabase.auth.onAuthStateChange().map((event) { return event.session?.user; }); }); // Current user provider (synchronous) final currentUserProvider = Provider<User?>((ref) { return supabase.auth.currentUser; }); // Current session provider final currentSessionProvider = Provider<Session?>((ref) { return supabase.auth.currentSession; }); class AuthNotifier extends StateNotifier<AsyncValue<User?>> { AuthNotifier() : super(const AsyncValue.loading()) { _loadInitialUser(); } void _loadInitialUser() { final user = supabase.auth.currentUser; state = AsyncValue.data(user); } // Email/Password Sign Up Future<void> signUp({ required String email, required String password, required String username, }) async { state = const AsyncValue.loading(); try { final response = await supabase.auth.signUp( email: email, password: password, data: { 'username': username, 'created_at': DateTime.now().toIso8601String(), }, ); state = AsyncValue.data(response.user); } catch (e) { state = AsyncValue.error(e.toString(), StackTrace.current); rethrow; } } // Email/Password Sign In Future<void> signIn({ required String email, required String password, }) async { state = const AsyncValue.loading(); try { final response = await supabase.auth.signInWithPassword( email: email, password: password, ); state = AsyncValue.data(response.user); } catch (e) { state = AsyncValue.error(e.toString(), StackTrace.current); rethrow; } } // Google Sign In Future<void> signInWithGoogle() async { state = const AsyncValue.loading(); try { final GoogleSignIn googleSignIn = GoogleSignIn( serverClientId: 'YOUR_GOOGLE_WEB_CLIENT_ID', ); final googleUser = await googleSignIn.signIn(); if (googleUser == null) { throw Exception('Google sign-in cancelled'); } final googleAuth = await googleUser.authentication; final idToken = googleAuth.idToken; final accessToken = googleAuth.accessToken; if (idToken == null) { throw Exception('No ID token found'); } final response = await supabase.auth.signInWithIdToken( provider: OAuthProvider.google, idToken: idToken, accessToken: accessToken, ); state = AsyncValue.data(response.user); } catch (e) { state = AsyncValue.error(e.toString(), StackTrace.current); rethrow; } } // Sign Out Future<void> signOut() async { state = const AsyncValue.loading(); try { // Sign out from Google as well if used final GoogleSignIn googleSignIn = GoogleSignIn(); await googleSignIn.signOut(); await supabase.auth.signOut(); state = const AsyncValue.data(null); } catch (e) { state = AsyncValue.error(e.toString(), StackTrace.current); rethrow; } } // Update user metadata Future<void> updateProfile({ required String username, }) async { state = const AsyncValue.loading(); try { final response = await supabase.auth.updateUser( data: {'username': username}, ); state = AsyncValue.data(response.user); } catch (e) { state = AsyncValue.error(e.toString(), StackTrace.current); rethrow; } } } final authNotifierProvider = StateNotifierProvider<AuthNotifier, AsyncValue<User?>>((ref) { return AuthNotifier(); });
