Photo Gallery with Animations

Difficulty: Medium Module: Advanced Flutter Concepts: Hero Animations, Grid Layout, Gestures, Infinite Scroll Est. Time: 40-50 min ⚡ Advanced

📋 Challenge Description

Build a Photo Gallery with smooth hero animations, grid layouts, and interactive gestures. Users can browse photos in a grid, tap to view with beautiful transitions, and pinch-to-zoom on individual images.

✅ Requirements

  • Display photos in a responsive grid layout
  • Use Hero animations for smooth transitions
  • Implement pinch-to-zoom on photo detail view
  • Add fade-in animations for images as they load
  • Show infinite scrolling with pagination
  • Display photo metadata (title, upload date)
  • Add search functionality with animated results
  • Support light/dark theme toggle
  • Use Riverpod for state management
  • Handle loading states and errors gracefully

🌐 API Endpoint

Unsplash API - Random Photos

https://api.unsplash.com/photos?page=1&per_page=20
💡 Note: You'll need to register for a free Unsplash API key at unsplash.com/developers

📚 Resources

  • http Package
  • Riverpod
  • Hero Animations Guide
  • InteractiveViewer for Pinch-to-Zoom
Need a hint? Click here

💡 Hint: Use Hero widget with a unique tag for each photo. For the grid, use SliverGrid with GridView.builder . Use InteractiveViewer for the detail view with pinch-to-zoom. For infinite scrolling, use ScrollController with pagination. Use FadeInImage or Image.network with a placeholder.
