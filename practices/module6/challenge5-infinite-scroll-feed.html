Infinite Scroll Feed

Difficulty: Medium Module: Advanced Flutter Concepts: Pagination, Scroll Controller, API Integration, Loading States Est. Time: 35-45 min ⚡ Advanced

📋 Challenge Description

Build an Infinite Scroll Feed that loads posts from the JSONPlaceholder API with pagination. The app should load more content as the user scrolls, with smooth loading indicators and error handling.

✅ Requirements

  • Fetch posts from JSONPlaceholder API with pagination
  • Implement infinite scrolling with scroll detection
  • Show loading indicators for initial and subsequent loads
  • Display post cards with title, body, and user info
  • Handle error states with retry functionality
  • Add pull-to-refresh to reload the feed
  • Show end-of-feed indicator when all posts loaded
  • Use Riverpod for state management
  • Support dark/light theme
  • Add search functionality to filter posts

🌐 API Endpoint

JSONPlaceholder - Posts

https://jsonplaceholder.typicode.com/posts?_page=1&_limit=20
💡 Note: JSONPlaceholder is a free mock API - no API key required!

📚 Resources

  • http Package
  • Riverpod
  • JSONPlaceholder
  • Flutter Pagination Guide
Need a hint? Click here

💡 Hint: Use ScrollController with addListener to detect when the user scrolls near the bottom. Track the current page number and use it in API requests. Use FutureProvider for loading posts and StateProvider for pagination state. Show a CircularProgressIndicator at the bottom of the list when loading more.
