Bookmark Manager

Difficulty: Easy Module: Local Storage Concepts: SharedPreferences, JSON Serialization, CRUD Est. Time: 25-35 min ⚙️ SharedPreferences

📋 Challenge Description

Build a Bookmark Manager app that allows users to save, organize, and manage their favorite URLs. All bookmarks should be stored locally using SharedPreferences .

✅ Requirements

  • Create a Bookmark model with id , title , url , category , timestamp
  • Use SharedPreferences for local storage
  • Store bookmarks as JSON strings
  • Use Riverpod for state management
  • Display a list of bookmarks sorted by timestamp
  • Allow users to add new bookmarks (title + URL)
  • Allow users to delete bookmarks
  • Add category support (e.g., Work, Personal, Study, Other)
  • Add search functionality to filter bookmarks
  • Handle empty state when no bookmarks exist
  • Add launch URL functionality (open in browser)

⚙️ SharedPreferences Setup

SharedPreferences is a simple key-value store for primitive data types.

flutter pub add shared_preferences
💡 Note: SharedPreferences is ideal for small amounts of data like user settings or bookmarks. For storing complex objects, convert them to JSON strings.

3. Storage Service (lib/services/storage_service.dart)

lib/services/storage_service.dart import 'dart:convert' ; import 'package:shared_preferences/shared_preferences.dart' ; import '../models/bookmark.dart' ; class StorageService { static const String _bookmarksKey = 'bookmarks' ; static SharedPreferences? _prefs; static Future < void > init () async { _prefs = await SharedPreferences. getInstance (); } static Future <List<Bookmark>> loadBookmarks () async { if (_prefs == null ) await init (); final String? jsonString = _prefs!. getString (_bookmarksKey); if (jsonString == null || jsonString. isEmpty ) { return []; } final List<dynamic> jsonList = json. decode (jsonString); return jsonList . map ((json) => Bookmark. fromJson (json)) . toList () .. sort ((a, b) => b.timestamp. compareTo (a.timestamp)); } static Future < void > saveBookmarks (List<Bookmark> bookmarks) async { if (_prefs == null ) await init (); final String jsonString = json. encode ( bookmarks. map ((bookmark) => bookmark. toJson ()). toList (), ); await _prefs!. setString (_bookmarksKey, jsonString); } static Future < void > clearAll () async { if (_prefs == null ) await init (); await _prefs!. remove (_bookmarksKey); } }
