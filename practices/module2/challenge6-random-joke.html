Random Joke App

Difficulty: Easy Module: API Integration Concepts: HTTP GET, JSON Parsing, Random Data Est. Time: 20-30 min 😂 Official Joke API

📋 Challenge Description

Build a Random Joke App that fetches random jokes from the Official Joke API. Users can tap a button to get a new joke, with a clean and entertaining UI.

🌐 API Endpoint

Official Joke API - Random Joke

https://official-joke-api.appspot.com/random_joke

Returns a random joke with setup, punchline, and type (general, programming, etc.).

💡 Note: No API key required! Free to use for learning.

✅ Requirements

  • Fetch a random joke from the API
  • Display the setup and punchline
  • Show the joke type (e.g., "General", "Programming")
  • Add a "New Joke" button to fetch another
  • Show a loading indicator while fetching
  • Handle errors gracefully
  • Use FutureProvider for state management
  • Use http package for API calls

📚 Resources

  • Official Joke API
  • http package
  • Flutter Networking

💡 Hint: Create a Joke model with type , setup , and punchline . Use FutureProvider.autoDispose with a refresh counter to trigger new fetches. When the user taps "New Joke", increment the counter to invalidate the provider.

1. pubspec.yaml

pubspec.yaml dependencies: flutter: sdk: flutter flutter_riverpod: ^2.5.1 http: ^1.1.0

2. Joke Model (lib/models/joke.dart)

lib/models/joke.dart class Joke { final String type; final String setup; final String punchline; Joke ({ required this .type, required this .setup, required this .punchline, }); factory Joke. fromJson (Map<String, dynamic> json) { return Joke( type: json[ 'type' ] as String, setup: json[ 'setup' ] as String, punchline: json[ 'punchline' ] as String, ); } }

3. Joke Provider (lib/providers/joke_provider.dart)

lib/providers/joke_provider.dart import 'package:flutter_riverpod/flutter_riverpod.dart' ; import 'package:http/http.dart' as http; import 'dart:convert' ; import '../models/joke.dart' ; final refreshCounterProvider = StateProvider< int >((ref) => 0 ); final jokeProvider = FutureProvider<Joke>.autoDispose((ref) async { // Watch refresh counter to trigger new fetches ref. watch (refreshCounterProvider); final url = Uri.parse( 'https://official-joke-api.appspot.com/random_joke' ); final response = await http. get (url); if (response.statusCode == 200 ) { final data = json. decode (response.body); return Joke. fromJson (data); } else { throw Exception( 'Failed to load joke: ${response.statusCode}' ); } });

4. Main Screen (lib/main.dart)

lib/main.dart import 'package:flutter/material.dart' ; import 'package:flutter_riverpod/flutter_riverpod.dart' ; import 'providers/joke_provider.dart' ; import 'widgets/joke_card.dart' ; void main () { runApp( const ProviderScope(child: MyApp())); } class MyApp extends StatelessWidget { const MyApp ({ super .key}); @override Widget build (BuildContext context) { return MaterialApp( title: 'Random Joke' , theme: ThemeData( colorScheme: ColorScheme.fromSeed(seedColor: Colors.amber), useMaterial3: true , ), home: const HomeScreen(), ); } } class HomeScreen extends ConsumerWidget { const HomeScreen ({ super .key}); void _refreshJoke (WidgetRef ref) { ref. read (refreshCounterProvider.notifier).state++; } @override Widget build (BuildContext context, WidgetRef ref) { final jokeAsync = ref. watch (jokeProvider); return Scaffold( appBar: AppBar( title: Text( '😂 Random Joke' ), backgroundColor: Colors.amber, foregroundColor: Colors.black, actions: [ IconButton( onPressed: () => _refreshJoke(ref), icon: Icon(Icons.refresh), tooltip: 'New Joke' , ), ], ), body: Center( child: Padding( padding: EdgeInsets. all ( 24 ), child: jokeAsync. when ( loading: () => Column( mainAxisAlignment: MainAxisAlignment.center, children: [ CircularProgressIndicator(), SizedBox(height: 16 ), Text( 'Loading a joke...' , style: TextStyle(color: Colors.grey[ 600 ]), ), ], ), error: (error, stack) => Column( mainAxisAlignment: MainAxisAlignment.center, children: [ Icon(Icons.error_outline, size: 64 , color: Colors.red[ 300 ]), SizedBox(height: 16 ), Text( 'Oops! Something went wrong' , style: TextStyle(fontSize: 18 , fontWeight: FontWeight.bold), ), SizedBox(height: 8 ), Text( error. toString (), textAlign: TextAlign.center, style: TextStyle(color: Colors.grey[ 600 ]), ), SizedBox(height: 16 ), ElevatedButton( onPressed: () => _refreshJoke(ref), child: Text( 'Try Again' ), ), ], ), data: (joke) => JokeCard(joke: joke, onNewJoke: () => _refreshJoke(ref)), ), ), ), ); } }

5. Joke Card Widget (lib/widgets/joke_card.dart)

FontWeight.bold : FontWeight.normal, color: _showPunchline ? Colors.black : Colors.grey[ 500 ], fontStyle: _showPunchline ? FontStyle.normal : FontStyle.italic, ), textAlign: TextAlign.center, ), ), ], ), ), ), SizedBox(height: 24 ), // New joke button SizedBox( width: double .infinity, child: ElevatedButton.icon( onPressed: () { setState (() => _showPunchline = false ); widget.onNewJoke(); }, icon: Icon(Icons.refresh), label: Text( 'New Joke' ), style: ElevatedButton.styleFrom( backgroundColor: Colors.amber, foregroundColor: Colors.black, padding: EdgeInsets. symmetric (vertical: 14 ), shape: RoundedRectangleBorder( borderRadius: BorderRadius. circular ( 8 ), ), ), ), ), SizedBox(height: 8 ), Text( '👆 Tap the punchline to reveal' , style: TextStyle( fontSize: 12 , color: Colors.grey[ 500 ], ), ), ], ), ), ); } }
