News Reader App

Difficulty: Medium Module: API Integration Concepts: HTTP GET, Categories, Search, Pagination Est. Time: 60-75 min 📰 NewsAPI

📋 Challenge Description

Build a News Reader App that fetches and displays news articles from the NewsAPI. Users should be able to browse news by categories, search for specific topics, and load more articles with pagination.

🌐 API Endpoints

NewsAPI - Top Headlines

https://newsapi.org/v2/top-headlines?country=us&category={category}&apiKey={API_KEY}&page={page}

NewsAPI - Everything (Search)

https://newsapi.org/v2/everything?q={query}&apiKey={API_KEY}&page={page}

Returns news articles with title, description, image, source, and published date.

⚠️ Note: You'll need a free API key from NewsAPI . The free tier allows 100 requests per day.

✅ Requirements

  • Display top headlines on app start
  • Category filtering (Business, Technology, Sports, Health, Entertainment)
  • Search for news articles by keyword
  • Pagination (load more articles)
  • Show article image , title , source , and time
  • Tap an article to open in browser
  • Handle loading and error states
  • Use http package for API calls
  • Use StateNotifier for state management

📚 Resources

  • NewsAPI Documentation
  • http package
  • url_launcher package
  • StateNotifierProvider

💡 Hint: Use StateNotifier to manage the news state with properties like articles , category , page , and hasMore . For pagination, use a ScrollController to detect when the user reaches the bottom of the list. Use url_launcher to open articles in the browser.

3. News Notifier (lib/providers/news_provider.dart)

lib/providers/news_provider.dart import 'package:flutter_riverpod/flutter_riverpod.dart' ; import 'package:http/http.dart' as http; import 'dart:convert' ; import '../models/article.dart' ; const String apiKey = 'YOUR_API_KEY_HERE' ; // Replace with your NewsAPI key class NewsState { final List<Article> articles; final String category; final int page; final bool hasMore; final bool isLoading; final String? error; final String searchQuery; NewsState ({ this .articles = const [], this .category = 'general' , this .page = 1 , this .hasMore = true , this .isLoading = false , this .error, this .searchQuery = '' , }); NewsState copyWith ({ List<Article>? articles, String? category, int ? page, bool ? hasMore, bool ? isLoading, String? error, String? searchQuery, }) { return NewsState( articles: articles ?? this .articles, category: category ?? this .category, page: page ?? this .page, hasMore: hasMore ?? this .hasMore, isLoading: isLoading ?? this .isLoading, error: error ?? this .error, searchQuery: searchQuery ?? this .searchQuery, ); } } class NewsNotifier extends StateNotifier<NewsState> { NewsNotifier () : super (NewsState()) { _fetchArticles (); } Future < void > _fetchArticles ({ bool loadMore = false }) async { if (state.isLoading) return ; if (loadMore && !state.hasMore) return ; state = state. copyWith (isLoading: true , error: null ); try { final page = loadMore ? state.page + 1 : 1 ; final isSearch = state.searchQuery. isNotEmpty ; final url = isSearch ? Uri.parse( 'https://newsapi.org/v2/everything?q=${state.searchQuery}&apiKey=$apiKey&page=$page&pageSize=20' , ) : Uri.parse( 'https://newsapi.org/v2/top-headlines?country=us&category=${state.category}&apiKey=$apiKey&page=$page&pageSize=20' , ); final response = await http. get (url); if (response.statusCode == 200 ) { final data = json. decode (response.body); final List<dynamic> results = data[ 'articles' ]; final newArticles = results . map ((json) => Article. fromJson (json)) . where ((article) => article.title != null && article.title!. isNotEmpty ) . toList (); state = state. copyWith ( articles: loadMore ? [...state.articles, ...newArticles] : newArticles, page: page, hasMore: newArticles. isNotEmpty , isLoading: false , error: null , ); } else { state = state. copyWith ( isLoading: false , error: 'Failed to load news: ${response.statusCode}' , ); } } catch (e) { state = state. copyWith ( isLoading: false , error: e. toString (), ); } } void loadMore () { _fetchArticles (loadMore: true ); } void changeCategory (String category) { state = state. copyWith ( category: category, searchQuery: '' , articles: [], page: 1 , hasMore: true , ); _fetchArticles (); } void search (String query) { if (query. trim (). isEmpty ) { state = state. copyWith (searchQuery: '' ); changeCategory (state.category); return ; } state = state. copyWith ( searchQuery: query, articles: [], page: 1 , hasMore: true , ); _fetchArticles (); } void refresh () { _fetchArticles (); } } final newsProvider = StateNotifierProvider<NewsNotifier, NewsState>((ref) { return NewsNotifier(); }); final categories = [ 'general' , 'business' , 'technology' , 'sports' , 'health' , 'entertainment' , 'science' , ];
