Currency Converter

Difficulty: Medium Module: API Integration Concepts: HTTP GET, JSON Parsing, Real-time Rates Est. Time: 45-60 min 💱 exchangerate.host

📋 Challenge Description

Build a Currency Converter app that fetches real-time exchange rates from the exchangerate.host API. Users can select currencies, enter an amount, and see the converted value instantly.

🌐 API Endpoint

exchangerate.host - Latest Rates

https://api.exchangerate.host/latest?base={base_currency}&symbols={target_currency}

Returns the latest exchange rate between two currencies. No API key required!

💡 Note: Free to use with 1500 requests per month. Supports over 170 currencies.

✅ Requirements

  • Select from and to currencies
  • Enter an amount to convert
  • Fetch real-time exchange rate from the API
  • Display the converted amount
  • Show the exchange rate used
  • Swap currencies with one tap
  • Handle loading and error states
  • Use http package for API calls
  • Use FutureProvider for state management

📚 Resources

  • exchangerate.host API
  • http package
  • Flutter Networking

💡 Hint: Use StateProvider for the base currency, target currency, and amount. Use FutureProvider.autoDispose with dependencies on these providers. The conversion calculation is: converted = amount * rate . Use a Provider to calculate the result from the rate and amount.

2. Currency Provider (lib/providers/currency_provider.dart)

lib/providers/currency_provider.dart import 'package:flutter_riverpod/flutter_riverpod.dart' ; import 'package:http/http.dart' as http; import 'dart:convert' ; // Available currencies final currencies = [ 'USD' , 'EUR' , 'GBP' , 'JPY' , 'CAD' , 'AUD' , 'CHF' , 'CNY' , 'INR' , 'BRL' , 'KRW' , 'MXN' , 'SGD' , 'NZD' , 'HKD' , 'SEK' , 'NOK' , 'DKK' , ]; // State providers final baseCurrencyProvider = StateProvider<String>((ref) => 'USD' ); final targetCurrencyProvider = StateProvider<String>((ref) => 'EUR' ); final amountProvider = StateProvider< double >((ref) => 1.0 ); // Rate provider final rateProvider = FutureProvider< double >.autoDispose((ref) async { final base = ref. watch (baseCurrencyProvider); final target = ref. watch (targetCurrencyProvider); final url = Uri.parse( 'https://api.exchangerate.host/latest?base=$base&symbols=$target' , ); final response = await http. get (url); if (response.statusCode == 200 ) { final data = json. decode (response.body); final rates = data[ 'rates' ] as Map<String, dynamic>; return (rates[target] as num ). toDouble (); } else { throw Exception( 'Failed to load exchange rate: ${response.statusCode}' ); } }); // Converted amount (derived state) final convertedAmountProvider = Provider< double >((ref) { final rate = ref. watch (rateProvider); final amount = ref. watch (amountProvider); return rate. when ( data: (rateValue) => amount * rateValue, loading: () => 0.0 , error: (_, __) => 0.0 , ); }); // Swap currencies final swapProvider = Provider<VoidCallback>((ref) { return () { final base = ref. read (baseCurrencyProvider); final target = ref. read (targetCurrencyProvider); ref. read (baseCurrencyProvider.notifier).state = target; ref. read (targetCurrencyProvider.notifier).state = base; }; });
