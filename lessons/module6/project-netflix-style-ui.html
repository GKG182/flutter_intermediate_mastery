Netflix Style UI

Build a stunning Netflix-inspired streaming app interface with slivers, custom scroll views, and hero animations.

UI Project Slivers Docs Hero Docs
Project Overview

In this project, you'll build a Netflix Style UI — a stunning streaming app interface with a collapsing header, horizontal scrolling rows, and smooth hero transitions. You'll master slivers, custom scroll views, and hero animations to create a polished, professional streaming experience.

📚 Prerequisites

  • Understanding of Slivers
  • Understanding of Hero Animations
  • Understanding of Custom Widgets
  • Familiarity with Animations

🎯 Learning Objectives

  • Master slivers for complex scrolling layouts
  • Build collapsing headers with SliverAppBar
  • Create horizontal scrolling content rows
  • Implement hero animations for content transitions
  • Build a Netflix-style UI with custom scroll views
  • Combine multiple sliver types for a polished experience

Sliver Components

1. Content Row (SliverList)

lib/widgets/content_row.dart class ContentRow extends StatelessWidget { final String title; final List<Movie> movies; const ContentRow({ required this.title, required this.movies, super.key, }); @override Widget build(BuildContext context) { return Column( crossAxisAlignment: CrossAxisAlignment.start, children: [ Padding( padding: EdgeInsets.symmetric(horizontal: 16, vertical: 8), child: Text( title, style: TextStyle( fontSize: 20, fontWeight: FontWeight.bold, color: Colors.white, ), ), ), SizedBox( height: 220, child: ListView.builder( scrollDirection: Axis.horizontal, padding: EdgeInsets.symmetric(horizontal: 8), itemCount: movies.length, itemBuilder: (context, index) { return Padding( padding: EdgeInsets.symmetric(horizontal: 8), child: MovieCard(movie: movies[index]), ); }, ), ), ], ); } }

2. Movie Card with Hero

lib/widgets/movie_card.dart class MovieCard extends StatelessWidget { final Movie movie; final bool isLarge; const MovieCard({ required this.movie, this.isLarge = false, super.key, }); double get cardWidth => isLarge ? 200 : 140; double get cardHeight => isLarge ? 280 : 200; @override Widget build(BuildContext context) { return GestureDetector( onTap: () { Navigator.push( context, MaterialPageRoute( builder: (context) => MovieDetailScreen(movie: movie), ), ); }, child: Hero( tag: 'movie_${movie.id}', child: ClipRRect( borderRadius: BorderRadius.circular(8), child: Container( width: cardWidth, height: cardHeight, decoration: BoxDecoration( image: DecorationImage( image: NetworkImage(movie.imageUrl), fit: BoxFit.cover, ), ), child: Container( decoration: BoxDecoration( gradient: LinearGradient( begin: Alignment.bottomCenter, end: Alignment.topCenter, colors: [ Colors.black54, Colors.transparent, ], ), ), child: Align( alignment: Alignment.bottomLeft, child: Padding( padding: EdgeInsets.all(8), child: Text( movie.title, style: TextStyle( color: Colors.white, fontSize: 12, fontWeight: FontWeight.bold, ), maxLines: 2, overflow: TextOverflow.ellipsis, ), ), ), ), ), ), ), ); } }

3. Hero Image for Detail Screen

lib/widgets/hero_image.dart class HeroImage extends StatelessWidget { final String imageUrl; final String tag; final double height; const HeroImage({ required this.imageUrl, required this.tag, this.height = 300, super.key, }); @override Widget build(BuildContext context) { return Hero( tag: tag, child: Container( height: height, width: double.infinity, decoration: BoxDecoration( image: DecorationImage( image: NetworkImage(imageUrl), fit: BoxFit.cover, ), ), child: Container( decoration: BoxDecoration( gradient: LinearGradient( begin: Alignment.bottomCenter, end: Alignment.topCenter, colors: [ Colors.black87, Colors.transparent, ], stops: [0.0, 0.4], ), ), ), ), ); } }
