📝 Project 1

Notes App

Build a complete notes app with Hive for local storage, SharedPreferences for settings, and offline-first functionality.

Hive package shared_preferences
📋 Requirements

What You'll Build

  • A notes app with full CRUD operations (Create, Read, Update, Delete)
  • Hive for persistent storage of notes
  • SharedPreferences for app settings (theme preference)
  • Search functionality – Search notes by title or content
  • Filter by color – Organize notes with color categories
  • Dark mode support – Persisted with SharedPreferences
  • Real-time updates – Using Hive's ValueListenableBuilder
  • Note details – View full note content

Learning Objectives

  • Implement Hive with custom type adapters
  • Use SharedPreferences for app settings
  • Master ValueListenableBuilder for real-time UI
  • Build a search and filter system
  • Handle CRUD operations with local storage

📖 How This Solution Works

Step-by-Step Explanation

  • 1. Models – The Note class uses Hive annotations with 8 fields including color, tags, and pin status.
  • 2. Hive Setup – Hive.initFlutter() , adapter registration, and box opening.
  • 3. Theme Provider – ThemeProvider with SharedPreferences for dark mode persistence.
  • 4. Notes Screen – Main UI with search, color filters, and grid layout.
  • 5. CRUD Operations – Create, Read, Update, and Delete notes with real-time UI updates.
  • 6. ValueListenableBuilder – Automatically updates UI when notes change.
  • 7. Pin & Tags – Pinned notes appear first; tags provide quick categorization.
Hive Storage
All notes are stored locally in a Hive box with type-safe adapters for custom objects.
Real-time UI
ValueListenableBuilder listens to Hive box changes and rebuilds the UI automatically.
Search & Filter
Search by title, content, or tags. Filter notes by color with visual color chips.
Theme Persistence
Dark mode preference is saved to SharedPreferences and applied at app startup.

🎯 Key Takeaway

This project demonstrates how to build a complete notes app using Hive for local storage and SharedPreferences for settings. You've learned to implement CRUD operations , search and filter , and real-time UI updates with ValueListenableBuilder . Hive makes local storage fast and enjoyable!
